## About Me

I’m a software engineer focused on **AI-native developer tools, backend systems, automation platforms, and data-driven products**.

I like building products that turn messy workflows into clear, usable systems: job search automation, contract operations intelligence, CI repair agents, lead discovery tools, email verification pipelines, and real-time analytics dashboards.

- Strongest areas: **Python, FastAPI, TypeScript, React, Next.js, automation, APIs, data pipelines, and LLM-assisted workflows**
- I enjoy building from **0 → working product**, not just writing isolated scripts
- I care about **reliability, explainability, clear UX, and practical engineering tradeoffs**
- Current focus: **AI agents, workflow automation, backend platforms, and intelligent dashboards**
